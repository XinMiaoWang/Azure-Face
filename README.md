# Azure-Face
功能介紹:  
1.Show Training Data: 顯示目前訓練過的圖片  
2.Create Person: 建立個人資料  
3.Identify: 辨識圖片中人物是否在資料庫中  
4.Show pic: 顯示指定圖片
